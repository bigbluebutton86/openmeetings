<plotstyle>

style for plot area

Attributes

fillcolor (default: 0xFFFFFF)

bgcolor for the plot area

 
linecolor

color of the border for the plot area

 
linesize (default: 0)

size of border of the plot area

 
opacity (default: 1)

opacity of plot background area ( not the opacity of dataseries )
